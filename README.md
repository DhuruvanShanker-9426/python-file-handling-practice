# 📂 Python File Handling Practice

## 📌 Overview

This repository contains practice implementations of Python File Handling concepts. It demonstrates how to work with external text files for reading, writing, and basic data processing using Python.

---

## 🚀 Concepts Covered

* Opening and closing files
* Reading file content using:

  * `read()`
  * `readlines()`
* Iterating through file lines
* Counting characters and lines
* Writing data to files
* File modes (`r`, `w`, `x`)
* Using `with open()` for safe file handling
* Handling file paths (including raw strings)
